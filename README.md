clclean
=======

* 以下の条件に当てはまる cloudinary の画像を消す
    * type={{ config.type }}
    * created_at が {{ config.moment }} 秒より古いもの


Configure
---------

see `config.json.dist`


How to use
----------

**Dry dun***

```sh
node index.js
```


**Delete**

```sh
NODE_ENV=production node index.js
```
